# 🧑‍🤝‍🧑 KinKeeper - Friend Management App

## 📌 Project Overview

KinKeeper is a modern friend management web application built with Next.js. It helps users keep track of their friends, monitor relationship status, and manage personal connections in an organized way.

---

## 🚀 Live Site

👉 https://kinkeeper-next-js-project-sieam.vercel.app/

---

## 🛠️ Technologies Used

* Next.js
* React.js
* Tailwind CSS
* DaisyUI
* Context API
* JSON (for mock data)

---

## ✨ Key Features

### 1. 👥 Friend List Management

View all friends in a clean and responsive card-based layout.

### 2. 📄 Dynamic Friend Details Page

Each friend has a dedicated details page with dynamic routing.

### 3. ⏳ Timeline Tracking

Track interactions and updates using a timeline system for each friend.

---

## 📂 Project Structure

* `/app` → Main application routes
* `/components` → Reusable UI components
* `/data` → JSON data files
* `/context` → Global state management

---

## 📦 Deployment

This project is deployed on Vercel.
